<h1 align="center">Sistema Bancário em Python</h1>

<p align="center">
  Sistema bancário desenvolvido em Python utilizando Programação Orientada a Objetos (POO).
</p>

<hr>

<h2>📖 Descrição</h2>

<p>
Este projeto consiste em um Sistema Bancário desenvolvido em Python utilizando os conceitos de
Programação Orientada a Objetos (POO). O sistema permite realizar operações bancárias básicas
de forma simples e organizada por meio de um menu interativo no terminal.
</p>

<p>
O objetivo do projeto é aplicar conceitos fundamentais da linguagem Python, como classes,
objetos, encapsulamento, modularização e manipulação de listas.
</p>

<hr>

<h2>⚙️ Funcionalidades</h2>

<ul>
  <li>💰 Depósito em conta</li>
  <li>🏧 Saque em conta</li>
  <li>📄 Consulta de extrato</li>
  <li>🔄 Transferência entre contas</li>
  <li>📋 Listagem de contas cadastradas</li>
  <li>📚 Consulta do histórico geral de transações</li>
  <li>🔍 Consulta do histórico de uma conta específica</li>
  <li>🚪 Encerramento do sistema</li>
</ul>

<hr>

<h2>🛠️ Tecnologias Utilizadas</h2>

<ul>
  <li>Python 3</li>
  <li>Programação Orientada a Objetos (POO)</li>
</ul>

<h2>📑 Arquivos do Projeto</h2>

<h3>conta.py</h3>
<ul>
  <li>Gerenciamento das contas bancárias.</li>
  <li>Operações de depósito, saque, transferência e consulta de saldo.</li>
</ul>

<h3>historico.py</h3>
<ul>
  <li>Armazena e gerencia o histórico das movimentações realizadas.</li>
</ul>

<h3>main.py</h3>
<ul>
  <li>Arquivo principal da aplicação.</li>
  <li>Responsável pelo menu interativo e controle do fluxo do sistema.</li>
</ul>

<h2>🚀 Como Executar</h2>
